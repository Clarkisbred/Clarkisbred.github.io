# BreDucky & Shrimpy 🦆🦐

## Setup
1. Place your `BreDucko.png` in this folder (same directory as index.html)
2. Edit `config.js` with your real Supabase/Groq keys
3. Open `index.html` in a browser (or serve with a local server)

## Bots
- **BreDucky** — cheerful yellow duck, your original boy 🦆
- **Shrimpy** — bratty girl shrimp with attitude 🦐💅

## Admin Access
- **Desktop:** Hold `Shift + Alt`, type `breduck`
- **Mobile:** Tap the 🦆 in the drawer 30 times
- **Password:** set in config.js (`ADMIN_PW`)

## Admin Features
- 👁 **Watch Conversations** — see live chat logs for both bots
- 🎭 **Take Over** — click "Take Over" in the spy panel to reply manually as the bot
- 🤖 **Release** — hand back control to the AI
- 📢 **Broadcast** — send a banner announcement to all visitors (via Supabase)
- 🎙️ **Mic input** — speak to the bots
- 🔊 **TTS** — bots speak replies aloud
